# 🧠 NLP & Machine Learning Text Classification

**Developed By:** Thasin Mahmud Rifat Bhuiyan

---

## 📌 Project Overview

This project focuses on text classification using Natural Language Processing (NLP) and Machine Learning techniques. It converts text data into numerical form and applies different models to predict categories.

---

## ⚙️ Techniques Used

- Bag of Words (BoW)
- TF-IDF Vectorization
- CountVectorizer (custom tokenizer)
- Cross-Validation (Stratified K-Fold)

---

## 🤖 Machine Learning Models

- Logistic Regression
- Linear Support Vector Machine (SVM)
- Random Forest Classifier

---

## 📊 Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

---

## ⚖️ Handling Imbalanced Data

Used `class_weight='balanced'` to improve performance on imbalanced datasets by giving more importance to minority classes.

---

## 📁 Workflow

1. Load dataset (Excel / CSV)
2. Preprocess text data
3. Convert text to numerical features (BoW / TF-IDF)
4. Split features (X) and labels (y)
5. Train models using cross-validation
6. Evaluate performance
7. Compare different models

---

## 🎯 Objective

To analyze and compare different NLP techniques and machine learning models for accurate text classification.

---

## 📌 Results

The project compares multiple models and selects the best-performing one based on evaluation metrics.
